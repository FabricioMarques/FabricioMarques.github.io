BootStrap

Autor: Fabricio Marques.

Aula 09/042014

Tipo de framework básico de desenvolvimento que fornece opcoes de personalização gráfica pronta para a facilitação na criação de websites. Unindo o CSS ao Javascript, promete fazer milagres com pouco tamanho. O objetivo principal e lógico do Bootstrap é a reducao do tempo de desenvolvimento de um website, seja ele uma página simples estática ou um grande portal dinâmico.
