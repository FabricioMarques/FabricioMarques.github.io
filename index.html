Criado conta no github.

Foi passado a instalacao do gitbash para gerenciamento do github.

Apresentado o sublime text como editor.

Executado alguns comandos basicos do gitbash.

1 - Navegar ate a pasta do projeto. cd NomePasta/

2 - git status Verifica status de alteracao ou possiveis erros.

3 - git add nomearquivo Adiciona o arquivo a ser atualizado ou adicionado no github.

4 - git commit -m "comentario" Adiciona um comentario referente a que e o arrquivo. Como por exemplo, atualizacoes, o que afetaram, o que foi implementado...

5 - git push Sincroniza com github.
